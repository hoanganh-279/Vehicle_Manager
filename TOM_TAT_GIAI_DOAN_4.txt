   ────────────────────────────────────────────────────────────────
   mkdir static\uploads

   ────────────────────────────────────────────────────────────────

   python app.py

5. Truy cập:
   ────────────────────────────────────────────────────────────────
   http://localhost:5000

GET  /video_feed
     → Stream video với detection overlay
     → Sử dụng: <img src="/video_feed">

POST /capture_plate
     → Chụp ảnh và nhận diện biển số
     → Returns: JSON với plate info + images

POST /release_camera
     → Giải phóng camera resources
     → Cleanup khi không dùng

❌ Camera không hoạt động:
   → Check camera index (0, 1, 2...)
   → Check permissions
   → Try different camera

❌ Tesseract error:
   → Verify installation
   → Update path in camera_alpr.py line 28
   → Restart app

❌ YOLO model error:
   → Run: python -c "from ultralytics import YOLO; YOLO('yolov8n.pt')"
   → Check internet connection

❌ Images không hiển thị:
   → Check static/uploads/ exists
   → Check file permissions
   → Verify database paths
