═══════════════════════════════════════════════════════════════════
  TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════

❌ Camera không hoạt động:
   ─────────────────────────────────────────────────────────────────
   • Check camera index (0, 1, 2...)
   • Check camera permissions
   • Try different camera in dropdown

❌ Tesseract không tìm thấy:
   ─────────────────────────────────────────────────────────────────
   • Verify installation path
   • Update path in camera_alpr.py line 28
   • Restart app after install

❌ YOLO model không download:
   ─────────────────────────────────────────────────────────────────
   python -c "from ultralytics import YOLO; YOLO('yolov8n.pt')"

❌ Ảnh không hiển thị:
   ─────────────────────────────────────────────────────────────────
   • Check static/uploads/ exists
   • Check file permissions
   • Check paths in database

❌ Lỗi encoding:
   ─────────────────────────────────────────────────────────────────
   • Verify database uses NVARCHAR (not VARCHAR)
   • Check UTF-16LE encoding in app.py
   • Verify collation: Vietnamese_CI_AS
